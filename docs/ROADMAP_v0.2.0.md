# Quant System v0.2.0 升级计划

## 目标
打造比旧系统更强大的分钟级量化交易系统

## 新增策略

### 1. 网格交易策略 (GridTrading)
**适用**: 震荡市场，比 MeanReversion 更激进
**原理**: 
- 自动在价格区间内布设买卖网格
- 每跌 X% 买入，每涨 X% 卖出
- 无限网格，循环套利

**参数**:
- 网格数量: 5-10 格
- 网格间距: 1-3%
- 每格仓位: 总资金/网格数

### 2. 马丁格尔策略 (Martingale)
**适用**: 趋势回调，高风险高收益
**原理**:
- 首次建仓 1x
- 每跌 Y% 加仓 2x
- 反弹 Z% 全部平仓
- 智能减仓控制总风险

**风险控制**:
- 最大加仓次数: 3 次
- 总仓位上限: 70%
- 强制止损: -10%

### 3. 动量突破策略 (MomentumBreakout)
**适用**: 分钟级快速波动
**原理**:
- 监测 5分钟/15分钟动量
- 成交量突增 + 价格突破 = 追入
- 追踪止损，吃完整段行情

### 4. 机器学习信号策略 (MLSignal)
**适用**: 全市场状态
**原理**:
- 基于历史数据训练简单模型
- 预测未来5分钟涨跌概率
- 概率 > 65% 才交易

### 5. 资金流向追踪 (SmartMoney)
**适用**: 识别主力动向
**原理**:
- 分析分钟级大单流入流出
- 追踪"聪明钱"动向
- 跟随主力操作

## 新增功能

### 1. 多策略并行
- 不再限制单一策略
- 多策略同时运行，取最强信号
- 策略权重动态调整

### 2. 自适应参数
- 根据市场波动率自动调整参数
- 高波动 → 放宽止损，缩小网格
- 低波动 → 收紧止损，扩大网格

### 3. 回测系统
- 历史数据回测
- 策略绩效对比
- 参数优化

### 4. 高级风控
- 波动率加权仓位
- 相关性矩阵（避免多币种同向波动）
- 黑天鹅熔断（瞬间暴跌 >10% 全平）

### 5. 智能报警
- 微信/飞书/邮件通知
- 大额盈亏提醒
- 系统异常报警
